Perfect實作內容：
1. Filter：請在**首頁**實作 Filter 功能，篩選類別至少包含**學業、人際、社團、快樂、生氣、難過**，Filter 後僅顯示出所選日記卡
2. 新增日記卡時，可以更改日記卡的日期。更改日期時必須檢查該日期是否為合法日期，若不合法（如：2022.02.30、2023.13.01 等）則無法儲存日記卡。
